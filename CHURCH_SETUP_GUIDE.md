## Quick Start Guide for Church Computer

### STEP 1: Download and Install Node.js
1. Open your web browser
2. Go to: **https://nodejs.org**
3. Click "Download" for the LTS version
4. Run the downloaded file
5. Follow the installation wizard (click "Next" for everything)
6. **RESTART YOUR COMPUTER** after installation

### STEP 2: Copy Website Files
1. Copy the entire `foursquare-ajebo-website` folder to your computer
2. Recommended location: `C:\Church\foursquare-website\`
3. Make sure all files are copied correctly

### STEP 3: Start the Website
1. Go to the website folder
2. **Double-click on `start-website.bat`**
3. Wait for the installation to complete (first time only)
4. The website will start automatically

### STEP 4: Open the Website
1. Open your web browser (Chrome, Firefox, or Edge)
2. Go to: **http://localhost:3000**
3. You should see the church website!

## üö® If Something Goes Wrong

### Website Won't Start
1. Make sure Node.js is installed
2. Restart your computer
3. Try running `start-website.bat` again

### Can't Access Website
1. Make sure the command window stays open
2. Check that you're going to **http://localhost:3000**
3. Try refreshing the browser page
